# 绪论

主线：马克思主义中国化

主题：中国化马克思主义

毛泽东思想和中国特色社会主义理论体系为什么要以马克思主义中国化为主线？

* 中国共产党为什么选择马克思主义：
  * 马克思主义是揭示人类社会发展规律的真理
  * 实践情况而言，是马列主义和毛泽东思想指导人们走出长夜建立新中国

* 为什么必须实现马克思主义中国化：
  * 是解决中国实际问题的客观需要
  * 是马克思主义理论发展的内在要求
* 什么是马克思主义中国化：
  * 马克思主义在指导中国革命、建设和改革的实践中实现具体化
  * 把中国革命、建设和改革的实践经验和历史经验上升为马克思主义理论
  * 把马克思主义根植于中国的优秀文化当中
